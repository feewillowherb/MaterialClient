新增一个SettingsEntity

这个实体用于存储系统配置。


配置项：
1.地磅设置(仅有一个):
    串口: string (COM3)
    波特率: string (9600)
    通讯方式: string (TF0)

2.摄像头(可以有多个):
    Name: string (camera_1)
    Ip: string (192.168.3.245)
    Port: string (8000)
    Channel: string (1)
    UserName: string (admin)
    Password: string (fdkj112233)

3.高拍仪(仅一个):
    UsbDevice:

4.车牌识别(可以有多个):
    Name: string (camera_1)
    Ip: string (192.168.3.245)
    Direction: Enum (In Or Out)


5.开机自动启动
    Enable




新增一个View页面用于用户设置上述这些配置
在AttendedWeighingWindow中可以弹出这个设置窗口。保存后重启所有设备
